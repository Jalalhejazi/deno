# Getting Started

```bash
git clone https://superusers-kursus@dev.azure.com/superusers-kursus/deno/_git/deno
cd deno

deno run --allow-net weather.ts
deno run --allow-all server.ts
deno run -A server.ts
deno run -A city-weather.ts
deno run -A crud.ts
deno run test.ts

```
